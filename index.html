2. Dasar Teori

HTML (HyperText Markup Language) merupakan bahasa markup untuk menyusun halaman web yang dapat dibaca oleh browser. Browser seperti Google Chrome atau Firefox akan menafsirkan kode HTML dan menampilkannya menjadi visual atau user interface.

Beberapa Elemen HTML yang Sering Digunakan:

Tag Fungsi
<h1> - <h6>Judul atau heading
<p>Paragraf teks
<a>Hyperlink
<img>Menampilkan gambar
<br>Ganti baris
<hr>Garis pemisah horizontal
<b>Teks tebal
<i>Teks miring
